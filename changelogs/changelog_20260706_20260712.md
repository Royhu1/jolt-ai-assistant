  (`False`) adds no column — verified byte-identical (`mass_cluster` / `mass_moving` equal)
  so the report pipeline and `finetune` are unaffected.

  `keep_tractor_only_label=True`, keeps the loaded (`mass_cluster.notna()`) line exactly as
