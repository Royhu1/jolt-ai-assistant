### Q: 实现 Case 3（Eq.29 model-based ratio）

**背景**：上一次提交的 ep_cruise_correction.py 只实现了 Case 1/2，Case 3 的 20 条 trip 没有计算
$EP_{cruise@90}$，导致 33% 样本丢失。用户决定继续推进但不辨识 $C_dA$。

**思路**：直接在 1 Hz 速度序列上计算两个理论量并取比值，避免显式加减速段识别：

$$EP_{c90}^{\text{Case 3}} = EP_{total}^{\text{obs}} \cdot
\frac{EP_{c90}^{\text{theo, flat}}}{EP_{total}^{\text{theo}}}$$

- 分子 $EP_{c90}^{\text{theo, flat}}$：Eq.22 with $\Delta h=0,\ v_c=25$ m/s（flat 90 km/h 纯巡航）
- 分母 $EP_{total}^{\text{theo}}$：在 1 Hz 上按 sample 积分 $\Delta KE + W_{res} + E_{grad}$，用 $\eta_{dt}$
  折算到电池端，减速时按 $\eta_{regen}$ 回收
- 比值即 $f_{no\_cruise}$（等价于 Eq.29 的 $EP_{total}/f$）
- 不需要再额外做 elevation correction，因为 $EP_{c90}^{\text{theo, flat}}$ 已经是平路参考

**参数决策**：

- $C_{rr} = 0.00465$, $C_dA = 6.16$（simulation baseline fallback）
- $\eta_{dt} = 0.90$, $\eta_{bat}(T)$ 用 Arrhenius
- **$\eta_{regen} = 0.42$**（YK73 实测系统级值，来自 `regen_analysis/report.md`；
  simulation baseline 0.90 是 ideal 上限）

**结果（60-trip 完整数据）**：

| Case | N | mean $EP_{c90}$ (kWh/km) | σ | R² vs Mass |
|:----:|:-:|:------------------------:|:-:|:----------:|
| Case 1 | 28 | 1.111 | 0.329 | 0.15 |
| Case 2 | 12 | 1.322 | 0.191 | 0.40 |
| Case 3 | 20 | **1.608** | 0.227 | **0.02** |

**Case 3 数值偏高 ~0.5 kWh/km** —— 是**方法学局限**不是代码 bug。

**诊断**：

- 对一条典型 Case 3 trip（低速长程 172 km, mean speed 54 km/h）：
  - $EP_{total}^{\text{obs}} = 1.38$ kWh/km
  - $EP_{total}^{\text{theo}} = 0.76$ kWh/km (with $\eta_{regen}=0.42$)
  - $EP_{c90}^{\text{theo}} = 1.00$ kWh/km
  - ratio = 1.00 / 0.76 = 1.32 → $EP_{c90}^{\text{corrected}} = 1.82$ kWh/km
- Theoretical EP 远低于 observed，说明 model **systematically 低估** observed EP
- 原因：**未建模的 overhead**（auxiliary 能耗、真实 $\eta_{dt}$ < 0.90、路况 $C_{rr}$ > 0.00465、
  行为噪声）在低速/启停工况中占比大，但在 pure 90 km/h 巡航中 negligible
- Eq.29 的 ratio 方法隐含"$k$ constant" 假设失效，把低速工况的 overhead 全部转嫁给 cruise

**结果物理意义**：

- Case 3 mean = 1.608 **不是**"这些 trip 的 EP_c90 estimate"
- 而是"按当前 model 参数，这些 trip 的 observed overhead scale 后的 cruise@90 等效"
- 要把 Case 3 数值和 Case 1/2 直接对齐，需要 calibrate model 参数（把 $\eta_{dt}$、$C_{rr}$ 等
  调整到让 Case 1 trip 的 $EP_{total}^{\text{theo}} \approx EP_{total}^{\text{obs}}$）

**report §3.5 新增"Case 3 的 model-dependent 偏差"小节** 详细说明这一点；推荐读者只用 Case 1+2
作为最终 ground truth，Case 3 作为方法学参考。

**图片更新**：

- `ec_ep_mass_per_case.png`：从 2 列（Case 1, Case 2）扩到 3 列，Case 3 子图标题加
  "(model-dependent)"
- `ec_case_distribution.png`：右侧直方图加入 Case 3（橙色），明显偏右
