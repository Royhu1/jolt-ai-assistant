Why vendored
------------
The dashboard detail pages (detail_<REG>.html) must open offline by double-click,
with no CDN / external network references. Both files are inlined verbatim into
every generated detail page (uPlot JS inside a <script>, CSS inside a <style>).

How to refresh
--------------
    python -m jolt_toolkit.report_generator.data_dashboard --fetch-uplot
This re-downloads both files from the jsDelivr URLs above into this directory.
After refreshing, update the version + sha256 lines here to match.
