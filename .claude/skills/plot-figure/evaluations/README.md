# plot-figure — evaluations

Per-run experience log (skill-design principle 7: accumulate experience). One file per
notable run: `<topic-or-REG>_<YYYYMMDD>.md`, following the pattern set by
`param-tuner/evaluations/`.

Record here when a run teaches something reusable, e.g.:

- a bespoke figure request that needed a non-obvious data filter or exclusion set;
- a workspace-routing decision (where the persisted script landed and why);
- a style-contract edge case (axis limits clipping real data, legend overflow);
- an output-format request and how it was resolved (see `references/output-formats.md`).

Routine standard-set regenerations do not need an entry.
